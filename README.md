# 👗 Clothing Price Comparison Website

## 📌 Project Overview

This project is a **Clothing Price Comparison Web Application** that allows users to search for clothing items and compare prices across multiple e-commerce platforms like Amazon, Flipkart, Myntra, and Meesho.

The goal of this project is to help users find the **best deals** by displaying product details and prices from different websites in one place.

---

## 🚀 Features

* 🔍 Search for clothing items (shirts, dresses, jeans, etc.)
* 💰 Compare prices across multiple platforms
* 🖼️ Display product image, name, and details
* ⭐ Show ratings and reviews (if available)
* 🔗 Direct link to purchase from original website
* 🎯 Highlight the lowest price
* 📱 Responsive and user-friendly UI
